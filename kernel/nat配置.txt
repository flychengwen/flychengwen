组网拓扑：
|--------------内部网络internet_aaa-------|----------内部网络internet_bbb-------------|
192.168.1.104 (vm) ------------>(192.168.1.24, 100.125.1.41) -----------> 100.125.1.65

目标：
tcp sync: 192.168.1.104 ->192.168.1.24

中间设备实现nat的步骤：
pre-routing: dnat     192.168.1.24 -> 100.125.1.65
fwd
post-routing: snat    192.168.1.104 -> 100.125.1.41


源：
ip addr add 192.168.1.104/24 dev eth1  


中间设备：
echo 1 > /proc/sys/net/ipv4/ip_forward
ip addr add 100.125.1.41/24 dev enp0s9
ip addr add 192.168.1.24/24 dev enp0s8
iptables -t nat -A PREROUTING -d 192.168.1.24 -j DNAT --to 100.125.1.65
iptables -t nat -A POSTROUTING -s 192.168.1.104 -j SNAT --to 100.125.1.41
